# Manual Document Dropzone

Add source files here before running the pipeline.

Recommended filenames are listed in `data/source_manifest.json`, for example:

- `accounts-parent-2025.pdf`
- `accounts-parent-2024.pdf`
- `accounts-parent-2023.pdf`
- `charge-0006.pdf`
- `charge-0005.pdf`

The pipeline can run before these files are present using the seeded Gail's dataset, but extracted text and table coverage will improve as the files are added.
